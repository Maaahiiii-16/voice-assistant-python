# 🎙️ AI Voice Assistant using Python

## 📌 Overview

This project is a **Python-based AI Voice Assistant** that can perform various tasks using voice commands such as searching Wikipedia, opening websites, checking weather updates, and answering computational queries.

---

## 🚀 Features

* 🎤 Speech Recognition (Voice Input)
* 🔊 Text-to-Speech Response
* 🌐 Open Websites (YouTube, Google, Gmail)
* 📚 Wikipedia Search
* 🌦️ Weather Information (via API)
* 📰 Latest News Headlines
* 📷 Capture Photos using Camera
* 🧠 Answer Questions using WolframAlpha

---

## 🛠️ Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* Wikipedia API
* OpenWeatherMap API
* WolframAlpha API
